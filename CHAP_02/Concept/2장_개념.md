## 🧠 2.1 퍼셉트론 (Perceptron) 상세 정리

### 1. 퍼셉트론과 인공 신경망의 탄생

- **역사적 배경**: 인공 신경망(ANN)은 사람의 신경망을 본떠 만든 알고리즘입니다. 1943년 워렌 스터기스 맥컬록(Warren Sturgis McCulloch)과 월터 피츠(Walter Pitts)가 사람의 뇌세포(뉴런)를 수학적으로 표현한 **퍼셉트론(Perceptron)** 을 처음 발표했습니다.
    
- **구조적 정의**: 퍼셉트론은 곧 '인공 뉴런'을 뜻하며, 이를 이용해 만든 모든 구조물을 인공 신경망이라고 부릅니다.
    
    - **단층 신경망**: 퍼셉트론이 단 하나만 존재하는 구조
        
    - **다층 신경망**: 여러 개의 퍼셉트론을 조합하여 층(Layer)을 쌓은 구조
        
---

### 2. 퍼셉트론의 동작 원리 (생물학적 뉴런과의 비교)

실제 뇌세포는 다른 뇌세포로부터 **일정 강도(임곗값) 이상의 자극(입력)**을 받지 않으면 다음 세포로 정보를 전달하지 않습니다. 인공 신경망 역시 이 원리를 그대로 차용하여, 입력값들을 가공한 뒤 **활성화 함수**를 거쳐 다음 노드로 정보를 넘길지 말지를 결정합니다.

퍼셉트론은 크게 **1) 입력층, 2) 노드(인공 뉴런), 3) 출력층**으로 구성되며 세부 요소는 다음과 같습니다.
![[Pasted image 20260220121234.png]]
- **입력값 ($x_1, x_2, \dots$)**: 모델에 들어오는 데이터 자극입니다.
    
- **가중치 ($w_1, w_2, \dots$)**: 입력에 대한 **중요도**를 나타내는 숫자입니다. 결과에 큰 영향을 미치는 중요한 입력값은 키우고, 중요하지 않은 입력값은 줄이는 역할을 합니다. (학습 중에는 고정된 상수로 취급됩니다.)
    
- **가중합 ($a$)**: 각각의 입력값에 대응하는 가중치를 곱하고 모두 더한 값입니다. 이 값이 활성화 함수의 입력으로 들어갑니다.
    
- **편향 ($b$, Bias)**: 활성화 함수가 작동하는 **임곗값을 좌우로 이동**시키는 역할을 합니다.
    
- **활성화 함수 ($F$)**: 임곗값을 기준으로 노드의 출력값을 최종적으로 결정하는 함수입니다. 이 함수를 거침으로써 신경망에 층을 쌓는 효과(비선형성)가 생집니다.
    
    ![[Pasted image 20260220121424.png]]
- **출력값 ($y$)**: 가중합이 활성화 함수를 통과하여 나온 최종 결과입니다.
    

퍼셉트론의 최종 출력 수식은 다음과 같이 표현됩니다.

$$y = F(x_1w_1 + x_2w_2 + b)$$

---

### 3. 활성화 함수: 시그모이드(Sigmoid) 함수를 쓰는 이유
    
![[Pasted image 20260220121648.png]]
- **특징**: 실수 전체의 모든 입력을 받아들여, 출력을 항상 **$0$과 $1$ 사이의 실수**로 변환해 주는 함수입니다.
    
- **사용 목적**: 딥러닝 모델의 출력값을 단순한 수치가 아닌 **확률(Probability)** 로써 다루고 해석하기 위함입니다. 확률 역시 $0$에서 $1$ 사이의 값으로 표현되기 때문에 시그모이드 함수가 적합하게 사용됩니다.
    

---

### 4. 결정 경계 (Decision Boundary)와 데이터 분류

퍼셉트론 내부의 가중합 수식인 $x_1w_1 + x_2w_2 + b = a$ 는 2차원 그래프상에서 **하나의 직선**으로 나타납니다.

- **결정 경계의 형성**: 가중치($w$)가 변함에 따라 이 직선의 기울기와 위치가 변합니다. 뉴런의 계산 결과인 $a$값에 따라 데이터가 이 직선의 위에 있는지, 아래에 있는지 판가름할 수 있으므로 이 직선을 **결정 경계**라고 부릅니다.
    ![[Pasted image 20260220122027.png]]
- **단층 신경망의 분류**: $0$과 $1$로 이루어진 데이터가 주어졌을 때, 적당한 가중치를 찾아 올바른 결정 경계(직선)를 긋는다면 데이터를 두 그룹으로 구분할 수 있습니다. 예를 들어 입력 데이터가 결정 경계 값보다 크면 $1$, 작으면 $0$으로 분류하는 식입니다.
    ![[Pasted image 20260220122119.png]]
- **의의**: 이처럼 하나의 결정 경계를 만들기 위해 단 하나의 퍼셉트론을 사용했기 때문에 이러한 구조를 **단층 신경망**에 의한 분류라고 합니다.

## 🚀 2.2 다층 신경망(MLP)으로 단층 신경망의 한계 극복하기

### 1. 단층 신경망의 치명적 한계: XOR 문제
![[Pasted image 20260220122312.png]]
단층 신경망은 2차원 평면상에서 단 하나의 직선(결정 경계)만을 그어 데이터를 분류합니다. 이 방식은 대부분의 단순한 논리 연산(AND, OR 등)을 해결할 수 있지만, **XOR(배타적 논리합)** 데이터는 절대로 분류할 수 없다는 치명적인 한계가 있습니다.

- **XOR 데이터의 특징**: 두 입력값($x_1$, $x_2$) 중 하나만 1일 때 정답이 1이 되고, 둘 다 같으면(0,0 또는 1,1) 정답이 0이 되는 데이터입니다.
    ![[Pasted image 20260220231457.png]]
- **불가능의 증명**: 이 데이터들을 좌표평면에 찍어보면, **어떤 방식으로 단 하나의 직선을 긋더라도 0과 1을 완벽하게 반으로 가를 수 없습니다.** (1969년 마빈 민스키가 수학적으로 증명)
    

---

### 2. 해결책: 다층 신경망과 기저 벡터 변환

직선 하나로 안 된다면 **직선을 여러 개(다층 구조)** 사용하면 됩니다. 다층 신경망이 XOR 문제를 푸는 핵심 원리는 **'기저 벡터 변환(Basis Vector Transformation)'**을 통해 데이터가 놓인 공간 자체를 구부리거나 재배치하는 것입니다.
![[Pasted image 20260220231521.png]]
-> - 은닉층은 데이터를 **새로운 축으로 투영**해서 표현한다.

**공간 변환 과정 (기하학적 직관)**

1. **기존 공간 (데이터 좌표평면)**: $x_1$, $x_2$ 축을 기준으로 데이터가 섞여 있어 직선 하나로 분류 불가.
    
2. **새로운 기준선 긋기 (은닉층의 역할)**: 기존 공간에 2개의 새로운 직선(결정 경계)을 긋습니다. 이 두 직선이 새로운 공간의 축(기저 벡터) 역할을 하게 됩니다.
    
3. **공간의 이동 (결정 경계 좌표평면)**: 기존 데이터 점들이 이 두 개의 새로운 직선(축)을 기준으로 다시 평가되어 새로운 좌표평면으로 이동합니다.
    
4. **최종 분류 (출력층의 역할)**: 새롭게 이동된 공간에서는 데이터들이 예쁘게 정렬되어, **단 하나의 직선**으로도 0과 1을 완벽하게 나눌 수 있게 됩니다.
    

---

### 3. 다층 신경망 (Multi-Layer Perceptron, MLP)의 구조

이처럼 공간을 변환하기 위해 여러 개의 퍼셉트론 층을 겹겹이 쌓은 구조를 다층 신경망이라고 합니다.

![[Pasted image 20260220232716.png]]

- **입력층 (Input Layer)**: 외부 데이터($x$)가 처음 들어오는 층입니다.
    
- **은닉층 (Hidden Layer)**: 입력층과 출력층 사이에 존재하는 층들입니다.
    
    - 이곳에서 위에서 설명한 '기저 벡터 변환(공간 왜곡)' 연산이 일어납니다.
        
    - 사용자가 굳이 이 층에서 계산되는 중간 결괏값을 직접 확인할 필요가 없기 때문에 '숨겨진(Hidden)' 층이라고 부릅니다.
        
- **출력층 (Output Layer)**: 최종적으로 변환된 데이터 공간에서 최종 결론(예측값)을 도출하는 층입니다.
    

---

### 💡 딥러닝 구조 용어 정리 & 실무 TIP

- **순전파 (Forward Propagation)**: 데이터가 입력층 $
ightarrow$ 은닉층 $
ightarrow$ 출력층 방향으로 흘러가며 계산되는 과정.
    
- **층의 깊이 (Depth)**: 은닉층이 몇 개나 깊게 쌓여 있는가. (이 깊이가 깊어질수록 '딥러닝'이라고 부릅니다.)
    
- **층의 너비 (Width)**: 하나의 은닉층 안에 노드(뉴런)가 몇 개나 있는가.
    
- **⚡ 실무 TIP (OOM 대응)**: 모델 학습 중 GPU 메모리가 부족하다면(Out Of Memory), 노드 수를 줄이는 것(너비 축소)보다 **층의 개수를 줄이는 것(깊이 축소)** 이 메모리 확보에 훨씬 더 효과적일 때가 많습니다. 각 층의 중간 계산 결과를 모두 메모리에 저장해야 하기 때문입니다.
    

- **층의 개수 = 층의 깊이** → 영어로는 **network depth** 라고 합니다.  
    (예: "a deep neural network"은 층이 많은 신경망을 뜻해요.)
- **노드의 수 = 층의 너비** → 영어로는 **network width** 라고 합니다.  
        (예: "wide neural network"은 각 층에 노드가 많은 신경망을 의미해요.)

## 🧮 2.3 인공 신경망의 학습 확인 (순전파 실습)

**순전파(Forward Propagation)** 란 데이터가 입력층 $
ightarrow$ 은닉층 $
ightarrow$ 출력층 방향으로 순서대로 전달되며 결괏값을 계산해 내는 과정을 말합니다.

### 1. 신경망 세팅 및 활성화 함수

실습을 위해 아래와 같은 가중치($w$)와 편향($b$)이 이미 완벽하게 세팅된 다층 신경망이 있다고 가정합니다.

- **입력층**: $x_1$, $x_2$
    ![[Pasted image 20260221012455.png]]
- **은닉층 노드 계산식**:
    ![[Pasted image 20260221012451.png]]
    - $a_1 = F(5x_1 + 5x_2 - 8)$
    - $a_2 = F(-7x_1 - 7x_2 + 3)$
        
- **출력층 노드 계산식**:
    $y' = F(w_5a_1 + w_6a_2 + b_3)$
    - $y' = F(-11a_1 - 11a_2 + 6)$
        

계산을 단순화하기 위해, 이번 예제에서는 가중합이 $0$ 이상이면 $1$을, $0$ 미만이면 $0$을 반환하는 아주 단순한 **계단 함수(Step Function)** 형태의 이진 분류 활성화 함수 $F(x)$를 모든 층에 동일하게 적용합니다.

$$F(x) = \begin{cases} 1 & (x \ge 0) \\ 0 & (x < 0) \end{cases}$$
![[Pasted image 20260221012717.png]]

---

### 2. XOR 데이터 순전파 계산 과정

XOR 문제의 4가지 입력 케이스에 대해 신경망이 어떤 값을 출력($y'$)하는지 직접 계산해 봅니다.

단층 신경망은 XOR를 구현하지 못함 -> 다중 신경망은 가능하다는 것을 증명

$a_1 = F(5x_1 + 5x_2 - 8) = F(5 \times 0 + 5 \times 0 - 8) = F(-8) = 0$  
$a_2 = F(-7x_1 - 7x_2 + 3) = F(-7 \times 0 - 7 \times 0 + 3) = F(+3) = 1$

![[Pasted image 20260221012919.png]]

| **입력값 ($x_1, x_2$)** | **실제 정답 ($y$)** | **1단계: 은닉층 $a_1$ 계산<br>$F(5x_1 + 5x_2 - 8)$** | **2단계: 은닉층 $a_2$ 계산<br>$F(-7x_1 - 7x_2 + 3)$** | **3단계: 최종 출력 $y'$ 계산<br>$F(-11a_1 - 11a_2 + 6)$** | **결과 확인** |
| :--- | :---: | :---: | :---: | :--- | :---: |
| **0, 0** | **0** | $F(-8) = 0$ | $F(3) = 1$ | $F(-11(0) - 11(1) + 6) = F(-5) = 0$ | **일치** ✅ |
| **0, 1** | **1** | $F(-3) = 0$ | $F(-4) = 0$ | $F(-11(0) - 11(0) + 6) = F(6) = 1$ | **일치** ✅ |
| **1, 0** | **1** | $F(-3) = 0$ | $F(-4) = 0$ | $F(-11(0) - 11(0) + 6) = F(6) = 1$ | **일치** ✅ |
| **1, 1** | **0** | $F(2) = 1$ | $F(-11) = 0$ | $F(-11(1) - 11(0) + 6) = F(-5) = 0$ | **일치** ✅ |

---

### 3. 결론

계산 결과, 다층 신경망의 최종 출력값인 $y'$가 실제 정답인 $y$(XOR 진리표)와 **완벽하게 일치**했습니다.

즉, **"제대로 된 가중치($w$)와 편향($b$) 값만 알아낼 수 있다면, 다층 신경망은 아무리 복잡한 데이터 분포(XOR 등)라도 정확히 분류해 낼 수 있다"**는 사실이 증명되었습니다.

## 🎯 2.4 손실 함수로 올바른 가중치 찾기

### 1. 결정 경계의 성능 평가와 손실 함수

데이터를 0과 1로 나누는 결정 경계(직선)는 무수히 많이 그을 수 있습니다. 그렇다면 그중 **"어떤 선이 가장 데이터를 잘 분류했는가?"** 를 평가할 객관적인 지표가 필요합니다.
![[Pasted image 20260221013302.png]]
- **오차(Error)의 정의**: 각각의 데이터 점으로부터 결정 경계(직선)까지의 거리.
    
- **손실 함수 (Loss Function)**: 이 오차들을 계산하여 모델의 성능을 수치화하는 함수입니다. (오차 함수라고도 부릅니다.)
    ![[Pasted image 20260221022745.png]]
- **핵심 원리**: 오차의 평균이 작을수록 데이터를 잘 분류한 것입니다. 즉, **손실 함수의 결괏값이 가장 작게 나오는 결정 경계가 가장 좋은 결정 경계**입니다.
    

---

### 2. 대표적인 손실 함수 종류

풀고자 하는 문제의 종류(회귀/분류)에 따라 알맞은 손실 함수를 선택해야 합니다.

|**함수명**|**용도**|**설명 및 특징**|
|---|---|---|
|**평균 제곱 오차**<br>(MSE, Mean Squared Error)|**회귀**|정답과 예측값 차이(오차)를 **제곱하여 평균** 낸 값.<br><br>오차가 1보다 작으면 더 작게, 1보다 크면 더 크게 증폭시켜 큰 오차에 민감하게 반응합니다.|
|**크로스 엔트로피 오차**<br>(CE, Cross Entropy Error)|**분류**<br>(이진/다중)|두 **확률 분포의 차이**를 구하는 함수.<br><br>분류 모델의 출력값은 '확률'이므로 이를 평가하기 위해 사용합니다. (정답값의 확률 $	imes$ 예측 확률의 로그값)|
|**평균 절대 오차**<br>(MAE, Mean Average Error)|**회귀**|정답과 예측값 차이의 **절댓값을 평균** 낸 값.<br><br>1보다 작은 미세한 오차도 놓치지 않지만, 오차의 크기보다는 부호에 의존하여 학습이 불안정해질 수 있습니다.|
|**평균 제곱근 오차**<br>(RMSE, Root Mean Squared Error)|**회귀**|MSE 값에 **제곱근($\sqrt{}$)**을 씌운 값.<br><br>MSE가 가진 단점인 '큰 오차에 대한 과도한 민감도'를 줄여줍니다.|

## ⚙️ 2.5 경사 하강법과 오차 역전파로 최적의 값 찾기 (도입)

### 1. 왜 새로운 방법론이 필요한가?

다층 신경망은 복잡한 데이터 분포를 학습하기 위해 은닉층의 깊이를 늘리거나 뉴런의 개수를 늘립니다.

하지만 모델이 복잡해질수록 찾아야 하는 가중치($w$)와 편향($b$)의 수가 기하급수적으로 증가합니다. 변수가 수만~수억 개로 늘어나면 전통적인 수학 방식(방정식을 직접 푸는 방식)으로는 오차가 가장 적은 최적의 선을 찾는 것이 불가능해집니다.

이를 해결하기 위해 딥러닝은 방정식을 푸는 대신, **경사 하강법(Gradient Descent)** 과 **오차 역전파(Backpropagation)** 라는 알고리즘을 사용하여 최적의 값을 점진적으로 찾아냅니다.
![[Pasted image 20260221023211.png]]
---

### 2. 신경망의 수학적 일반화 (변수화)

경사 하강법을 적용하기 위해, 이전 실습에서 사용했던 '고정된 숫자'들을 모두 가중치($w$)와 편향($b$)이라는 '변수'로 바꾸어 수식을 일반화합니다.

- **은닉층의 출력 계산 ($a$)**
    
    입력값($x$)에 각각의 가중치와 편향을 연산한 후, 활성화 함수($F$)를 통과시킵니다.
    $$a_1 = F(w_1x_1 + w_2x_2 + b_1)$$
    $$a_2 = F(w_3x_1 + w_4x_2 + b_2)$$
    
- **출력층의 최종 예측값 계산 ($y$)**
    
    은닉층의 결괏값($a$)을 다시 입력으로 받아 최종 출력을 계산합니다.
    $$y = F(w_5a_1 + w_6a_2 + b_3)$$
    

---

### 3. 최종 목표: 손실 함수의 최소화

이제 수식으로 표현된 신경망의 예측값($y$)과, 우리가 이미 알고 있는 실제 정답($y'$)을 비교하여 오차를 계산하는 **손실 함수($L$)** 를 다음과 같이 정의할 수 있습니다.

$$L(y, y')$$

결국 딥러닝 모델 학습의 최종 목표는 **이 손실 함수 $L$의 결과가 최소가 되도록 만드는 모든 가중치($w_1 \sim w_6$)와 편향($b_1 \sim b_3$)의 값을 찾아내는 것**입니다.

---

## 📉 2.5.1 경사 하강법 (Gradient Descent) 핵심 요약

### 1. 경사 하강법의 개념

- **정의**: 손실 함수(Loss Function)를 미분하여 구한 **기울기(경사)의 반대 방향**으로 가중치를 조금씩 이동시키며, 손실이 가장 적어지는 **최솟값(최적화 지점)**을 찾아가는 알고리즘입니다.
    
- **이동 방향의 원리**:
    ![[Pasted image 20260221095223.png]]
    - **음(-)의 기울기**일 때: 가중치를 **양(+)의 방향**으로 이동시켜야 최솟값에 가까워집니다.
    - **양(+)의 기울기**일 때: 가중치를 **음(-)의 방향**으로 이동시켜야 최솟값에 가까워집니다.
        

---

### 2. 가중치 업데이트 수식

경사 하강법을 통해 새로운 가중치($w'$)를 구하는 수식은 다음과 같습니다.

$$w' = w - \alpha \frac{\partial L(y, y')}{\partial w}$$

- $w'$ : 새롭게 업데이트된 가중치
- $w$ : 현재의 가중치
- $\alpha$ (알파) : **학습률 (Learning Rate)**
- $\frac{\partial L(y, y')}{\partial w}$ : 가중치 $w$에 대한 손실 함수 $L$의 미분값 (현재 위치에서의 **기울기**)
- **보폭 (Step Size)**: 수식에서 $\alpha 	imes 기울기$ 부분이 실제로 가중치가 한 번에 이동하는 거리(보폭)를 의미합니다.
    

---

### 3. 학습률 (Learning Rate, $\alpha$)의 중요성

기울기에 곱해져서 이동하는 크기(보폭)를 결정하는 아주 중요한 지표입니다. 적절한 학습률을 찾는 것이 모델 학습의 핵심입니다.
![[Pasted image 20260221095426.png]]
- **학습률이 너무 클 때**: 보폭이 너무 넓어서 최솟값을 찾지 못하고 계곡을 이리저리 건너뛰며 튕겨 나갈 위험이 있습니다. (발산)
- **학습률이 너무 작을 때**: 보폭이 너무 좁아서 최솟값에 도달하기까지 지나치게 오랜 시간이 걸리고 반복 횟수가 많아집니다. (학습 지연)
- **해결책**: 처음부터 완벽한 값을 알 수는 없으므로, 여러 번의 실험을 통해 반복적으로 학습률을 바꿔가며 **최적의 학습률**을 직접 찾아내야 합니다.

---

## 🔄 2.5.2 오차 역전파 (Error Backpropagation)

### 1. 오차 역전파의 핵심 개념

- **정의**: 정답과 신경망이 예측한 값 사이의 **오차(Error)**를 출력층에서부터 거꾸로 입력층 방향으로 흘려보내며 가중치를 수정하는 알고리즘입니다.
- **방향**: 데이터가 입력에서 출력으로 향하는 **순전파(Forward Propagation)와 정확히 반대 방향**으로 정보(오차)가 전달되기 때문에 '역전파'라고 부릅니다.
- **업데이트 순서**: 출력층에 가장 가까운 가중치부터 먼저 수정하고, 점차 입력층에 가까운 가중치들을 수정해 나갑니다.
    

---

### 2. 수학적 원리: 미분의 연쇄 법칙 (Chain Rule)
![[Pasted image 20260221104502.png]]
- **연쇄 법칙의 적용**: 출력층에서 구한 오차의 기울기를 이전 층으로 넘겨줄 때, **합성함수의 미분(미분의 연쇄 법칙)** 을 사용합니다.
- **기울기 누적 곱셈**: 층을 거꾸로 거슬러 올라갈수록, 앞서 계산된 기울기 값들이 계속해서 누적되어 곱해집니다.
- **결과**: 입력층에 가까운 가중치일수록 수많은 기울기들이 누적 곱셈된 결과를 바탕으로 새로운 가중치 값을 얻게 됩니다.
    

---

### 3. 💡 핵심 용어 리마인드

- **보폭 (Step Size)**: 가중치가 한 번에 이동하는 거리.
    - **보폭 = 학습률 $	imes$ 기울기**
    - 즉, 기울기가 가파를수록 보폭도 커지고, 기울기가 완만할수록 보폭도 작아집니다.

## 🛡️ 2.6 활성화 함수와 기울기 소실 예방

### 1. 기울기 소실 (Vanishing Gradient) 문제란?

오차 역전파 과정에서는 출력층에서 입력층으로 오차(기울기)를 전달할 때, 각 층을 지날 때마다 활성화 함수의 미분값이 계속해서 곱해집니다.

- **시그모이드(Sigmoid) 함수의 치명적 한계**:
![[Pasted image 20260221113311.png]]
시그모이드 함수를 미분한 도함수의 최댓값은 겨우 **0.25**입니다. 층을 거칠 때마다 0.25 이하의 소수들이 계속 곱해지기 때문에, 은닉층이 깊어질수록 전달되는 오차의 크기가 기하급수적으로 줄어들어 결국 **0에 수렴**하게 됩니다.

- **결과**: 출력층 근처의 가중치들은 잘 업데이트되지만, **입력층에 가까운 앞단의 은닉층들은 기울기가 전달되지 않아 학습이 아예 멈춰버리는 현상**이 발생합니다. 이것이 바로 기울기 소실 문제입니다.
    

### 2. 해결책: ReLU (렐루) 함수의 등장

기울기 소실을 막으려면 아무리 여러 번 미분값을 곱해도 값이 줄어들지 않는 새로운 활성화 함수가 필요했고, 그렇게 도입된 것이 **ReLU(Rectified Linear Unit)** 입니다.
![[Pasted image 20260221113416.png]]
- **동작 원리**: 입력값이 0보다 작으면 **0**을 반환하고, 0보다 크면 **입력값 그대로**를 반환합니다.
- **기울기 소실 예방**: 0보다 큰 범위에서는 미분값(기울기)이 항상 **1**입니다. 1은 아무리 여러 번 곱해도 값이 작아지지 않으므로, 층이 깊어져도 오차가 입력층까지 무사히 전달됩니다.
- **주의점 (Dying ReLU)**: 0보다 작은 값이 들어오면 출력과 미분값이 모두 0이 되어 뉴런이 죽어버리는 현상이 생길 수 있으므로 설계 시 주의가 필요합니다.

## 💡 2.7 활성화 함수의 역할과 종류

### 1. 활성화 함수가 없다면? (선형성의 한계)

활성화 함수를 쓰지 않고 가중치($W$)를 곱하고 편향($b$)을 더하는 연산만 반복한다면, 은닉층을 아무리 쌓아도 결국 '단층 신경망'과 수학적으로 동일해집니다.

- $a_1 = W_1x + b_1$
- $y = W_2a_1 + b_2$
- $y = W_2(W_1x + b_1) + b_2 = (W_2W_1)x + (W_2b_1 + b_2)$

즉, 수식을 정리하면 **$y = W'x + b'$** 가 되어, 처음부터 층을 하나만 둔 것과 아무런 차이가 없어집니다.

### 2. XOR 문제와 비선형성의 마법

단층 신경망(직선 1개)으로는 절대 XOR 데이터를 분류할 수 없습니다. 활성화 함수가 만들어내는 **'비선형성(Non-linearity)'**이 있어야만 데이터 공간을 왜곡하여 복잡한 데이터를 분류할 수 있게 됩니다.

### 3. ReLU의 진짜 역할은 '꺾임'에 있습니다

ReLU는 $x > 0$ 구간에서 기울기를 1로 유지해 **'기울기 소실'을 막으면서도**, $x < 0$일 때 값을 0으로 만들어버리는 **'꺾임'**을 통해 모델에 비선형성을 부여합니다. 이 꺾이는 지점 덕분에 모델은 복잡한 데이터 공간을 유연하게 학습할 수 있습니다.

### 4. 딥러닝 필수 활성화 함수 3가지 총정리

| **시그모이드**<br>(Sigmoid) | 입력값을 0과 1 사이의 실수로 압축하여 확률로 표현합니다.![[Pasted image 20260221120207.png]] | 출력층<br>(**이진 분류**) | `nn.Sigmoid()` |
