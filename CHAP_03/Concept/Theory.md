# 🧠 CHAP_03 이론 정리 (Concept)

## 1. 딥러닝 모델의 가중치(Weight)
- **가중치(Parameter):** 인공 신경망이 학습을 통해 스스로 조절하는 미지수입니다.
- **규모:** 이미지 인식 모델은 천만~1억 개, GPT-3와 같은 거대 모델은 약 1,750억 개의 가중치를 가집니다.
- **학습의 본질:** 수많은 가중치를 최적화하여 입력 데이터에 대한 정확한 정답을 찾아가는 과정입니다.

## 2. 딥러닝 학습 루프 (Training Loop)
1. **순전파 (Forward):** 데이터를 입력하여 모델의 예측값을 계산합니다.
2. **오차(Loss) 계산:** 예측값과 정답을 손실 함수(Loss Function)로 비교합니다.
3. **역전파 (Backpropagation):** 오차를 줄이기 위해 출력층에서 입력층 방향으로 기울기(Gradient)를 전달합니다.
4. **가중치 업데이트:** 계산된 기울기를 바탕으로 최적화(Optimizer) 알고리즘이 가중치를 수정합니다.

## 3. 다층 신경망 (MLP, Multi-Layer Perceptron)
- **특징(Feature):** 결과 예측에 사용되는 데이터 요소입니다.
- **완전연결층(FC Layer):** 이전 층의 모든 노드와 다음 층의 모든 노드가 1:1로 연결된 구조입니다.
- **활성화 함수(ReLU):** 선형 계산 결과에 비선형성을 부여하여 복잡한 패턴을 학습하게 합니다.

## 4. 학습 단위 (Batch, Epoch, Iteration)
- **배치(Batch):** 한 번에 업데이트할 때 사용하는 데이터 묶음입니다.
- **에포크(Epoch):** 전체 데이터를 한 번 다 훑는 단위입니다.
- **이터레이션(Iteration):** 1에포크를 위해 필요한 배치의 반복 횟수입니다.

## 5. 회귀(Regression) vs 분류(Classification)
- **회귀:** 신경망의 출력값을 그대로 사용 (예: 집값 예측). MSE(평균 제곱 오차)를 주로 사용합니다.
- **분류:** 출력값을 소프트맥스(Softmax) 함수를 통해 확률 분포로 변환 (예: 숫자 분류). CrossEntropyLoss를 사용합니다.
- **소프트맥스(Softmax):** 모든 출력값을 0~1 사이로 제한하고 총합을 1로 만들어 확률로 해석 가능하게 합니다.

## 6. 이미지 데이터 처리
- **Flatten (평탄화):** 2D 이미지(28x28)를 MLP 입력에 맞게 1D 벡터(784)로 변환하는 과정입니다.
- **ToTensor:** 이미지를 파이토치 텐서로 변환하며 픽셀값을 0~1 사이로 정규화합니다.
