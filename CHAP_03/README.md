# 📘 CHAP_03: 실습 첫걸음 (회귀와 다중분류)

이 장에서는 딥러닝의 기초인 회귀와 다중분류 문제를 파이토치로 해결하는 방법을 학습합니다.

## 📂 구성
- [🧠 이론 정리 (Concept)](Concept/Theory.md): 가중치, MLP, 학습 단위, 소프트맥스 등 핵심 개념
- [💻 실습 구현 (Code)](Code/Implementation.md): 사인 함수, 집값 예측, MNIST 분류 전체 코드

---

## 🎯 주요 실습 내용
### 1. 사인 함수 예측하기
- 3차 다항식을 이용한 곡선 근사 ($y = ax^3 + bx^2 + cx + d$)
- 수동 미분을 통한 오차 역전파 및 가중치 업데이트 원리 파악

### 2. 캘리포니아 집값 예측하기 (회귀)
- MLP(다층 신경망) 구조 설계
- MSE(평균 제곱 오차)를 이용한 수치 예측 실습

### 3. MNIST 손글씨 분류하기 (다중 분류)
- 2D 이미지의 Flatten(평탄화) 처리
- 소프트맥스 함수와 CrossEntropyLoss를 이용한 다중 클래스 분류
- GPU 학습 환경 설정 및 모델 저장/불러오기
